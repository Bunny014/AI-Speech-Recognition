# 🎤 AI Speech Recognition

## 📌 Project Overview

AI Speech Recognition is a web application built using **Python**, **Streamlit**, and **OpenAI Whisper**.

Users can upload an audio file and convert speech into text with high accuracy.

---

##  Features

- Upload audio files
- Supports MP3, WAV and M4A
- Speech-to-Text conversion using OpenAI Whisper
- Display transcription on screen
- Download transcript
- Save transcript automatically
- Processing time display
- Simple and clean Streamlit UI

##  Installation

Clone the repository

```bash
git clone https://github.com/Bunny014/AI-Speech-Recognition.git
```

Go to project folder

```bash
cd AI-Speech-Recognition
```

Create virtual environment

```bash
python -m venv .venv
```

Activate virtual environment

**macOS/Linux**

```bash
source .venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

##  Run the Project

```bash
streamlit run app.py
```
